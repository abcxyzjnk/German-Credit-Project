# German Credit Prediction Report

## Objective
Build a machine learning model that predicts credit risk for applicants using the German Credit dataset.

## Approach
- Explore and clean input features.
- Encode categorical fields for model training.
- Train baseline and tuned classification models.
- Compare metrics such as accuracy, precision, recall, and F1-score.

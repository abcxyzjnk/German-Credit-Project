# German-Credit-Project

This project focuses on credit risk prediction using the German Credit dataset.

## Workflow

- **Exploratory Data Analysis (EDA):** Investigate data quality, feature distributions, class balance, and key relationships between predictors and credit risk.
- **Model Selection:** Train and compare **three models** for credit risk classification.
- **Cross-Validation:** Evaluate model performance using cross-validation to get reliable, generalizable results and reduce overfitting risk.
